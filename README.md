# PerfectBlu

PerfectBlu is a lightweight, source-first knowledge base for the DUNE universe.

The project begins with the six Frank Herbert novels:

1. Dune
2. Dune Messiah
3. Children of Dune
4. God Emperor of Dune
5. Heretics of Dune
6. Chapterhouse: Dune

After the core six, the project can expand to Brian Herbert and Kevin J. Anderson novels, comics, films, television, games, and other media while keeping continuity boundaries clear.

The name PerfectBlu alludes to the Eyes of Ibad: blue within blue within blue.

Current direction: build a static website with structured Markdown or MDX content, strong citations, and minimal JavaScript.

Project documentation:

- `docs/plans/2026-05-24-perfectblu-project-plan.md`
- `docs/development.md`
- `docs/deployment.md`

Local build:

```bash
cd site
npm ci
npm run check
npm run build
```
